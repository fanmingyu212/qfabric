qFabric is a python package for controlling multiple AWGs using a universal interface designed for quantum physics experiments.

This package is still in development. The private and public APIs may change.

### Installation

For users of `qFabric`, run `pip install git+https://github.com/fanmingyu212/qfabric`.

If you want to develop `qFabric`, you can fork this project, clone the fork, navigate to the unzipped folder (containing `pyproject.toml`), and run `pip install -e .[dev,docs]` to install it in the editable mode with development dependencies.

See https://fanmingyu212.github.io/qfabric/ for documentation.
